## Подключённая библиотека: Day.js

**Репозиторий:** https://github.com/iamkun/dayjs  
**CDN:** https://cdnjs.cloudflare.com/ajax/libs/dayjs/1.11.10/dayjs.min.js

### Что такое Day.js?

Day.js — это минималистичная JavaScript-библиотека для работы с датами и временем (весит ~2 КБ против ~70 КБ у Moment.js), с совместимым API. Поддерживает плагины, локали, форматирование, парсинг и операции с датами.

В проекте подключены три файла:
```html
<!-- Ядро (включает английскую локаль по умолчанию) -->
<script src=".../dayjs.min.js"></script>
<!-- Локаль для русского языка -->
<script src=".../locale/ru.min.js"></script>
<!-- Локаль для немецкого языка -->
<script src=".../locale/de.min.js"></script>
```

### Где посмотреть работу библиотеки?

**Живые часы в шапке страницы** — компонент `#headerClock` в правой части хедера показывает текущие дату и время, обновляясь каждую секунду. Day.js используется для:

1. **Форматирования даты и времени** — метод `dayjs().format(...)` с разными шаблонами для каждого языка:
   - RU: `пятница, 13 июня • 14:32:05`  
   - EN: `Friday, June 13 • 14:32:05`  
   - DE: `Freitag, 13. Juni • 14:32:05`

2. **Переключения локали** — при смене языка интерфейса через кнопки 🇷🇺 / 🇬🇧 / 🇩🇪 вызывается `dayjs.locale('ru' | 'en' | 'de')`, после чего названия дней недели и месяцев в часах меняются автоматически.

Код находится в разделе **«10. Day.js — живые дата и время в хедере»** файла `script.js`.

## Переключение языков (i18n)

Реализовано без сторонних библиотек — на чистом JavaScript.

**Поддерживаемые языки:** Русский 🇷🇺, English 🇬🇧, Deutsch 🇩🇪

Кнопки-флаги расположены в шапке страницы. При клике переводятся все статичные тексты интерфейса: заголовки, лейблы полей, плейсхолдеры, тексты кнопок, подсказки, условия использования — без перезагрузки страницы и без изменения значений, введённых пользователем.
